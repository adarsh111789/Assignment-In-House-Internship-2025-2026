# Assignment 17 - PySpark DataFrame Application

## Objective
Analyze sales data using PySpark DataFrames.

Operations:
- Sort products by sales descending
- Display top 3 products
- Filter products with sales > 80000
- Save filtered output as CSV

## Build Docker Image
```bash
docker build -t assignment17 .
```

## Run Docker Container
```bash
docker run assignment17
```
